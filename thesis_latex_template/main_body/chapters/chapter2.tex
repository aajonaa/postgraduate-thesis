\chapter{相关理论及技术}

本章梳理后续研究所涉及的理论与技术背景，内容涵盖元启发式算法机制、应用场景建模、代理辅助优化策略以及统计分析方法，为后续算法设计与实验论证提供必要的理论基础。

\section{元启发式优化算法}

元启发式算法（MAs）是处理复杂优化任务的重要工具，在应对大规模搜索空间和缺乏显式解结构的NP-hard问题时具有较强的适应性\cite{ibsa_ref4,sbo_ref1}。按照设计灵感的不同，这类方法大致可分为群体智能、进化计算、自然规律模拟和人类行为模仿等类型\cite{sbo_ref11}。尽管具体更新机制存在差异，但其共同目标在于协调探索（Exploration）与开发（Exploitation），在扩大搜索覆盖面和加强局部精细搜索之间取得平衡\cite{ibsa_ref75}。

\subsection{群体智能与进化算法}

群体智能算法通过个体间的信息共享与协同更新实现全局搜索\cite{sbo_ref18}。粒子群优化（PSO）\cite{ibsa_ref26}是其中最具代表性的方法之一，其基本更新规则如下：
\begin{equation}
v_i^{t+1} = \omega v_i^t + c_1 r_1 (pbest_i - x_i^t) + c_2 r_2 (gbest - x_i^t)
\end{equation}
\begin{equation}
x_i^{t+1} = x_i^t + v_i^{t+1}
\end{equation}
其中，$\omega$为惯性权重，$c_1$和$c_2$为学习因子，$r_1$和$r_2$为$[0,1]$上的随机数，$pbest_i$和$gbest$分别表示个体历史最优位置与群体全局最优位置。由于机理清晰、实现简便，PSO已广泛用于函数优化、神经网络训练等连续优化相关任务；经过适当的离散化或编码设计后，也可用于部分组合优化问题。除PSO外，灰狼优化算法（GWO）\cite{ibsa_ref81}通过模拟灰狼的等级结构和围捕行为开展群体搜索，蚁群优化算法（ACO）\cite{ibsa_ref25}借助信息素沉积机制求解离散优化问题，人工蜂群算法（ABC）\cite{ibsa_ref27}则通过模拟蜜蜂觅食实现全局寻优。这些方法通过不同的信息传递与反馈机制，共同构成了群体智能搜索的基本方法体系。

与模拟群体行为的方法不同，进化算法更侧重于对遗传变异规律的工程模拟\cite{sbo_ref11}。差分进化（DE）\cite{ibsa_ref34}结构简洁，在实值优化任务中表现稳定，已成为代理辅助优化中的常用搜索工具。常用的DE/rand/1变异策略通过向量差分驱动搜索：
\begin{equation}
v_i = x_{r1} + F \cdot (x_{r2} - x_{r3})
\end{equation}
其中，缩放因子$F$与交叉概率$CR$共同调节变异幅度和种群多样性。该机制在保持较好收敛稳定性的同时\cite{ibsa_ref55}，也为后续章节中的算法设计提供了参考。

\subsection{回溯搜索算法}

回溯搜索算法（BSA）由Civicioglu\cite{ibsa_ref53}于2013年提出，是一种利用历史种群记忆进行搜索的进化算法。其主要流程包括初始化、Selection-I、变异、交叉和Selection-II五个阶段，其中变异策略利用当前种群与历史种群之间的差异生成候选解：
\begin{equation}
Mutant = X + F \cdot (OldX - X)
\end{equation}
其中，$F = 3 \cdot \operatorname{randn}$。历史记忆机制增强了算法的全局探索潜力，但由于变异与交叉过程具有较强的随机性，BSA在局部精细搜索方面仍存在不足\cite{ibsa_ref63}。Zhao\cite{ibsa_ref57}和Nama\cite{ibsa_ref59}等学者提出的改进方案，为第三章中IBSA算法的设计提供了重要参考。

\subsection{探索与开发平衡性}
\label{sec:ch2_balance_diversity}

为定量分析算法在搜索过程中的动态行为，本文引入基于种群多样性的度量指标。维度多样性$\text{Div}(FEs)$定义为：
\begin{equation}
\text{Div}(FEs)=\frac{1}{D}\sum^D_{j=1}{\frac{1}{N}\sum^N_{i=1}{\left|\operatorname{median}\left(X^j\left(FEs\right)\right)-X^j_i\left(FEs\right)\right|}}
\end{equation}

在此基础上，探索率与开发率分别定义为：
\begin{equation}
\text{Exploration}=\frac{\text{Div}\left(FEs\right)}{\text{Div}_{\max}}\cdot 100\%
\end{equation}
\begin{equation}
\text{Exploitation}=\frac{\left|\text{Div}\left(FEs\right)-\text{Div}_{\max}\right|}{\text{Div}_{\max}}\cdot 100\%
\end{equation}
进一步地，基于种群质心的欧氏距离多样性$I_C(FEs)$为：
\begin{equation}
I_C(FEs) = \sqrt{\sum^{N}_{i=1}{\sum^{D}_{j=1}{{\left(X^j_i\left(FEs\right)-C^j(FEs)\right)}^{2}}}}
\end{equation}
\begin{equation}
C^j(FEs) = \frac{1}{N}\sum^N_{i=1}{X^j_i\left(FEs\right)}
\end{equation}
$I_C(FEs)$越大，说明种群整体分散程度越高，算法更偏向于探索；$I_C(FEs)$较小时，种群逐渐聚集，算法则更偏向于开发。理想的寻优过程通常要求算法在前期保持较强的种群多样性，并在后期逐步向极值点收敛。后续章节将利用上述指标量化评估算法的动态搜索行为。

\section{特征选择与图像分割问题简介}

本文关注的高维特征选择与图像分割，本质上都可以表述为特定搜索空间中的参数寻优问题。前者需要在离散组合空间中筛选最优特征子集，后者则需要在连续灰度空间中确定最优分割阈值。

\subsection{特征选择问题定义}

特征选择（FS）通过剔除冗余维度，在简化模型结构的同时提升其任务表征能力\cite{ibsa_ref50}。主流方法包括过滤式、嵌入式和包裹式三类\cite{sbo_ref61}：过滤式依赖统计量进行独立评价，虽然效率较高，但难以刻画特征间的交互关系\cite{sbo_ref62}；嵌入式将选择过程整合进学习器训练；包裹式则以分类器性能为反馈导向，尽管计算开销较大，但得到的特征子集通常具有更强的判别能力\cite{ibsa_ref23,sbo_ref64}。

在基于元启发式的包裹式特征选择中，每个特征对应一个0/1决策变量，问题可视为二值优化。由于多数元启发式算法面向连续空间，通常需借助V型传递函数完成二值化：
\begin{equation}
T(x) = \left| \operatorname{erf}\left(\frac{\sqrt{\pi}}{2} x\right) \right|
\end{equation}
若$T(x_i^d) > \operatorname{rand}$，则对应维度取值翻转，否则保持不变。为同时兼顾分类误差和特征压缩率，适应度函数定义如下：
\begin{equation}
\text{Fitness}=\alpha\cdot \gamma_R\left(N\right)+\beta\cdot \frac{\left|R\right|}{\left|N\right|}
\end{equation}
后续实验将从平均适应度、分类错误率、所选特征数和运行时间四个方面评估算法表现。

\subsection{多阈值图像分割问题}

多阈值图像分割通过设置多个阈值将图像灰度空间划分为若干区域，是图像分割中的常见预处理步骤\cite{sbo_ref79,sbo_ref80}。若图像灰度级为$L$，则阈值向量可表示为：
\begin{equation}
\boldsymbol{T} = [t_1, t_2, \ldots, t_K], \quad 0 \le t_1 < t_2 < \cdots < t_K \le L-1
\end{equation}
其中$K$为阈值个数。随着$K$的增加，候选解空间会迅速膨胀，传统穷举方法难以在可接受时间内获得满意结果。采用元启发式算法在此类大规模搜索空间中高效搜索最优阈值组合，已成为该领域的主流做法\cite{sbo_ref82}。第四章将引入Renyi熵准则评估分割质量，并利用所提算法搜索最优阈值配置，以提高乳腺癌组织学图像的分割精细度\cite{sbo_ref83}。

\section{代理辅助昂贵优化技术}

在昂贵优化问题（EOPs）中，单次目标函数评估通常依赖高保真仿真或复杂工程计算，成本极高\cite{liu2023airfoils,kumar2024optimization}。代理辅助进化算法（SAEAs）通过低成本模型逼近真实目标函数，从而减少真实评估次数\cite{he2023review,jin2018data}，这也为第五章的算法设计提供了理论基础。

\subsection{代理模型与采样准则}
\label{sec:ch2_surrogate_models}

代理模型用于学习真实目标函数的近似映射关系\cite{kudela2022recent}。径向基函数（RBF）模型是一种基于距离的插值方法\cite{regis2013evolutionary}，其预测值可表示为已知样本点基函数值的加权组合：
\begin{equation}
\hat{f}(x) = \sum_{i=1}^{n} \lambda_i \phi(\|x - x_i\|)
\end{equation}
由插值条件$\hat{f}(x_i) = f(x_i)$可得矩阵形式：
\begin{equation}
\mathcal{F} = \boldsymbol{\Psi} \mathcal{W}
\end{equation}
其中$\boldsymbol{\Psi}_{ij} = \phi(\|x_i - x_j\|)$。在实际应用中，RBF常用的基函数包括多二次、反多二次、高斯和薄板样条等形式，不同基函数会影响模型的平滑性、插值精度与外推能力\cite{forrester2008engineering,kudela2022recent}。由于训练速度快，且对高维非线性分布具有较好的拟合能力\cite{kudela2022recent}，RBF被选作第五章中的基础代理模型。

图\ref{fig:ch2_RBF_prediction}展示了利用六个均匀分布样本对一维函数进行RBF近似的结果\cite{forrester2008engineering}。其中，白色圆圈表示样本点，黑色实线代表真实目标函数曲线，红色虚线代表RBF预测函数。从图中可以看出，RBF模型能够在样本点处实现精确插值，并在样本点之间给出平滑近似。

Kriging模型（高斯过程模型）\cite{buche2005accelerating,liu2013gaussian}除给出预测值外，还能提供不确定性估计，因此常与采样决策结合使用。该模型通常通过选择高斯、指数或Mat\'ern等相关函数刻画样本间的空间关联，并结合似然估计确定相应超参数；其详细推导与参数求解过程可参见相关文献\cite{buche2005accelerating,liu2013gaussian,forrester2008engineering}。为从候选解中筛选最值得进行真实评估的样本，通常需要引入采样准则\cite{zhan2020expected}。期望改进（EI）\cite{jones1998efficient}定义为：
\begin{equation}
\operatorname{EI}(x) = (f_{min} - \hat{f}(x)) \Phi\left(\frac{f_{min} - \hat{f}(x)}{s(x)}\right) + s(x) \phi\left(\frac{f_{min} - \hat{f}(x)}{s(x)}\right)
\end{equation}
其中，$f_{min}$为当前最优值，$\hat{f}(x)$和$s(x)$分别为代理模型的预测均值和标准差，$\Phi$和$\phi$分别为标准正态分布的累积分布函数和概率密度函数。置信下界（LCB）则定义为：
\begin{equation}
\operatorname{LCB}(x) = \hat{f}(x) - \kappa \cdot s(x)
\end{equation}
其中，$\kappa$越大，策略越倾向于探索。EI和LCB都体现了对探索与开发的综合权衡，使代理辅助优化能够在有限评估预算下做出更合理的采样决策。

\subsection{多臂赌博机策略}

多臂赌博机（MAB）问题描述了决策者如何在多个候选动作之间权衡已知高收益选择与未知探索\cite{slivkins2019introduction}，其核心在于处理探索与开发的平衡。在优化算法中，可以将不同搜索算子视为不同的臂，并根据环境反馈自适应调整选择概率\cite{xie2023surrogate,gong2024surrogate}。

上置信界（UCB）策略是解决MAB问题的经典方法之一\cite{auer2002finite}，其在函数评估次数为$FEs$时选择具有最高上置信界的臂：
\begin{equation}
a_{FEs} = \argmax_{a} \left[ Q_{FEs}(a) + c \sqrt{\frac{\ln FEs}{N_{FEs}(a)}} \right]
\end{equation}
其中，$Q_{FEs}(a)$为臂$a$的估计收益均值，$N_{FEs}(a)$为臂$a$被选择的次数，$c$为探索系数。UCB策略的关键在于：$Q_{FEs}(a)$鼓励选择历史表现较好的臂，而探索项$c \sqrt{\ln FEs / N_{FEs}(a)}$则鼓励尝试被选次数较少的臂。第五章将在此基础上构建赌博机驱动的自适应搜索机制，通过将异构搜索模式映射为不同的臂，实现搜索策略的动态分配。

\section{基准测试与统计检验}

\subsection{基准测试函数}

基准测试函数为算法性能评估提供了统一且可复现的比较环境\cite{ibsa_ref4}。IEEE CEC 2017测试套件包含30个函数，覆盖单峰（F1--F3）、多峰（F4--F10）、混合（F11--F20）和组合（F21--F30）四类问题\cite{ibsa_ref88}，能够从不同维度评估算法的搜索能力，是第三章和第四章的主要测评基准。IEEE CEC 2022测试套件面向函数评估受限的昂贵优化场景，包含12个函数\cite{kumar2024optimization}，最大评估次数受到严格限制，本文在第五章中利用其评估BEXEA的性能。

\subsection{统计检验方法}
\label{sec:ch2_stat_tests}

为保证实验比较的统计有效性，本文采用非参数统计检验对算法结果进行分析。Wilcoxon秩和检验\cite{ibsa_ref95}是一种成对比较方法，通过分析两组独立样本的排名差异判断二者差异是否具有统计显著性。Friedman检验\cite{derrac2011practical}则适用于多组比较，可同时评价多个算法在多个测试函数上的整体排名差异。在后文的实验分析中，符号‘+’、‘=’与‘-’分别表示算法在显著性水平$\alpha = 0.05$下优于、等同或劣于对比方法。
